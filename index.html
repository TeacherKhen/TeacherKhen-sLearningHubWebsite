🎮 How to Utilize the Games

  1. Each interactive game uses a custom .json file to store its structure, data, and your specific progress.
  2. If you want to use your game data on another device, make sure to first click Export Data inside the game dashboard and save that generated .json file.
  3. Then, simply open this website layout on your target new device, run the game, and load the .json file right back into it to resume!
